# Changelog

All notable changes to Kleos will be documented in this file.

## [1.0.0] — 2026-05-27

### Added
- **Floor Sweeper**: Single and batch floor buying, multi-collection sweep, auto-profit listing, stop-loss, undercut protection
- **Mint Sniper**: Mint discovery, pre-mint research checklist, auto-mint, post-mint monitoring, gas-aware minting
- **Portfolio Manager**: PnL tracking per NFT/collection, floor alerts, bulk listing, offer management, watchlist, portfolio rebalancing
- **Airdrop Checker**: On-chain eligibility scanning via contract queries + OpenSea API + Etherscan, multi-wallet support, merkle proof verification, scam detection
- **Batch Transfers**: Multi-NFT transfers to addresses, ENS, Twitter handles
- **Memory System**: Persistent logging to `/.memory/nft-activity.md`, watchlist, airdrop tracker, profit targets
- **Automation Presets**: 70+ copy-paste ready Bankr automation commands
- **Visual Branding**: Logo (geometric owl + hexagon), mascot, banner, social preview image
- **Repository**: README, CONTRIBUTING.md, issue templates, MIT license
