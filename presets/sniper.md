# 🎯 Mint Sniper — Copy-Paste Automations

Copy any of these commands directly into your Bankr agent. Replace placeholders with your values.

## Discovery

```
what's minting today?
```

```
show featured NFT mints
```

```
trending NFT collections
```

## Recurring Mint Discovery

```
every day at 8am, check what's minting today and show me promising drops
```

```
every morning, show me new NFT collections launching in the next 24 hours
```

```
every 6 hours, check upcoming mints on Base and Ethereum
```

## Pre-Mint Research

```
research [collection] before I mint — check supply, team, social presence, and legitimacy
```

```
is [collection] a legit project? Check their Twitter, Discord, and contract verification
```

```
compare [collection A] vs [collection B] — which has better mint economics?
```

## Mint Execution

```
mint from [manifold/seadrop link]
```

```
mint 3 from [collection link]
```

## Auto-Mint Setup

```
monitor [collection] mint page at [URL] and alert me when it goes live
```

```
every 30 minutes, check if [collection] mint is live at [URL]. If live, mint 2 NFTs
```

```
Set up auto-mint for [collection]:
1. Check [mint URL] every 30 minutes starting [date/time]
2. If mint is live, mint 2 NFTs
3. After minting, monitor floor every 15 minutes for 2 hours
4. Alert me if floor drops below mint price or exceeds 2x
```

## Post-Mint Monitoring

```
every 30 minutes for the next 24 hours, check [collection] floor price
```

```
monitor [collection] after my mint — alert me if floor goes below mint price
```

```
every hour, show me [collection] floor and volume since my mint
```

## Reveal Alerts

```
alert me when [collection] reveals and show the new floor price
```

```
every day at 12pm, check if [collection] has revealed yet
```

## Gas-Aware Minting

```
how much are gas fees on Ethereum right now?
```

```
check gas on Base before I mint — warn me if it's unusually high
```

## Combo Setup (Pro)

```
Set up mint sniper for [collection]:
1. Check [mint URL] every 30 minutes
2. When live, mint 3 NFTs
3. After mint, monitor floor every 15 minutes for 3 hours
4. If floor > 2x mint, list 1 NFT at floor
5. If floor < 0.8x mint, alert me to decide
6. At 24h mark, send full post-mint summary
```

> 💡 **Limits**: 5 automations (free) / 20 automations (Bankr Club). Mint discovery via browser automation requires Bankr Club or Max Mode.
