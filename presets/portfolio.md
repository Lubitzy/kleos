# 📊 Portfolio Manager — Copy-Paste Automations

Copy any of these commands directly into your Bankr agent.

## Portfolio Overview

```
show my NFT portfolio
```

```
what's my NFT portfolio worth?
```

```
show my NFTs on base
```

```
what NFTs do I own across all chains?
```

## Collection Breakdown

```
show my [collection] position
```

```
how are my [collection] NFTs doing?
```

```
show PnL for [collection]
```

```
every 6 hours, show my NFT portfolio PnL summary
```

```
every day at 9am, show my NFT portfolio with 24h changes
```

```
every Monday morning, send a weekly NFT portfolio report with PnL per collection
```

```
Set up NFT portfolio management:
1. Every 6 hours, show portfolio PnL summary
2. Every hour, check floors of all held collections
```
