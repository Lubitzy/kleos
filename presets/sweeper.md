# 🧹 Floor Sweeper — Copy-Paste Automations

Copy any of these commands directly into your Bankr agent. Replace `[collection]`, `[X]`, etc. with your values.

## Basic Floor Buy

```
buy the cheapest [collection]
```

```
sweep 3 floor [collection] on base
```

```
buy the 5 cheapest [collection] NFTs
```

## Multi-Collection Sweep

```
sweep the floor of [collection A], [collection B], and [collection C]
```

```
buy 1 floor from each collection on my watchlist
```

## Auto-Profit Listing

```
buy the cheapest [collection] and immediately list it for 2x the floor price
```

```
sweep 3 floor [collection] and list each at 50% profit
```

## Floor Price Alerts

```
every hour, check the floor price of [collection] and alert me if it drops 10%
```

```
every 30 minutes, check [collection] floor. If it dropped more than 15% from yesterday, alert me immediately
```

```
every 2 hours, show me [collection] floor price and 24h change
```

## Stop Loss

```
set a stop loss on my [collection] — if floor drops below [X] ETH, list all mine at floor immediately
```

```
every 15 minutes, check [collection] floor. If floor dropped more than 20% from my buy price, list all my [collection] NFTs at current floor
```

## Undercut Protection

```
every 30 minutes, check the floor price of [collection]. If my listing is above floor by more than 5%, cancel and relist at floor
```

```
keep my [collection] listings at the lowest price — relist if undercut
```

## Profit Taking

```
every hour, check my [collection] PnL. Alert me if I'm up more than 50% from buy price
```

```
when [collection] floor reaches [X] ETH, list all my [collection] NFTs at floor
```

## Watchlist Monitoring

```
every 4 hours, check floor prices for all collections in my watchlist and show changes
```

```
every day at 8am, check watchlist floors and highlight any that dropped more than 10%
```

## Combo Setup (Pro)

```
Set up NFT floor monitoring:
1. Every hour, check floor of [collection A], [collection B], [collection C]
2. Alert me if any floor drops 15% from yesterday
3. If any floor pumps 30%, alert me to consider selling
4. At 9pm daily, show summary of all floor movements
```

> 💡 **Limits**: 5 automations (free) / 20 automations (Bankr Club). Each execution counts toward daily runs (100 free / 1,000 Club).
